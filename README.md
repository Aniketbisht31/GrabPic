# GrabPic AI Event Photo Finder

---

 AI Event Photo Finder

AI-powered application that helps event attendees quickly find their photos using facial recognition.

 Overview

After events, organizers upload hundreds or thousands of photos. This app automatically detects faces in uploaded images and allows users to upload a selfie to instantly retrieve all photos where they appear.

 How GrabPic Works

* Organizer uploads event photos.
* AI detects and indexes faces.
* User logs in and uploads a selfie.
* System matches the face and returns relevant photos.

 Tech Stack

* **Frontend:** Flutter / React / Next.js
* **Backend:** FastAPI / Node.js
* **AI/ML:** OpenCV, InsightFace / DeepFace, FAISS
* **Database & Storage:** MongoDB / Firebase / AWS S3

 Features

* Automatic face detection
* Fast photo search using AI
* Private event spaces
* Scalable cloud storage

 Use Cases

College fests, weddings, hackathons, corporate events, and conferences.
